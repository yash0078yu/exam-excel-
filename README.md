Project Title: Retail Sales Analysis using Excel

Description:
This project is based on a retail sales dataset containing transaction details such as date, city, customer type, gender, product, category, quantity, price, and payment method. The objective is to analyze sales performance and generate useful business insights using Excel.

Dataset Details:
The dataset includes the following columns:

* Bill Number
* Date
* City
* Customer Type
* Gender
* Product
* Category
* Quantity
* Unit Price
* Total Amount
* Payment Method

Objectives:

* Analyze total sales and number of transactions
* Compare sales across different cities and categories
* Identify top-selling products
* Understand customer behavior (Member vs Normal)
* Analyze preferred payment methods

Tools and Techniques Used:

* Microsoft Excel
* Data Cleaning (removing duplicates, formatting)
* Formulas (SUM, IF, VLOOKUP, INDEX-MATCH, COUNTIF)
* Pivot Tables
* Charts (Bar, Pie, Column, Line)
* Slicers for interactive dashboard

Key Analysis:

* City-wise and category-wise sales
* Product performance based on quantity sold
* Payment method distribution
* Gender-based and customer-type-based analysis
* Sales trends over time

Conclusion:
This project helps in understanding sales patterns, customer preferences, and product performance. It can support decision-making for improving business strategies.
